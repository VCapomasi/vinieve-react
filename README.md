# ViNieve - Proyecto Final React

ViNieve es una tienda online especializada en productos para disfrutar la nieve con comodidad, abrigo y seguridad. Nuestra propuesta combina catálogo técnico, estética limpia y experiencia de compra simple.

Este proyecto final integra React, React Router, Context API, carrito, autenticación simulada, gestión de productos, cupones, búsqueda y diseño responsivo.

## Funcionalidades

- Catálogo de 10 productos.
- Vista de detalle de producto.
- Carrito de compras con Context API.
- Agregar productos al carrito.
- Modificar cantidades.
- Eliminar productos solo desde el carrito.
- Vaciar carrito.
- Cupón de descuento:
  - `VINIEVE10`
  - `NIEVE20`
- Modal simple de Mi cuenta.
- Usuario administrador de prueba.
- Gestión simple de productos: agregar y editar, sin borrar productos.
- Página Nosotros con información de la tienda y equipo.
- Diseño responsivo.

## Instalación local

```bash
npm install
npm run dev
```

## Build para Netlify

```bash
npm run build
```

Luego subir a Netlify la carpeta:

```text
dist
```
